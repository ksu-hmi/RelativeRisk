# RelativeRisk
Retrieve user input of demographic information as well as a series of yes/no or t/f responses to determine if user is at risk of acquiring the human immunodeficiency virus based on demographics such as age, sex, education, and other social determinants of health including behaviors. Additionally, will state recommendations, dietary changes, and appointment required.
